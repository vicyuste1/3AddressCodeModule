How to use


At the moment the app does not support subexpressions between parenthesis. In spite of this, if an expression with subexpressionssi is used there will be no erros; they will be transformed into its equivalent temporals and will be shown at the end of the translation.

This app allows to translate simple boolean to its equivalent 3-address code. Introducing text into the text field and clicking the translate button, the equivalent translation will appear in the bottom window.

Throughtthe menu "File" it is possible to save the transtlation, which will be saved in the root folder of th App.


Expressions format

Any expression introduced must follow the folowing format:

if (EXPRESSION) then EXPRESSION else EXPRESSION , where EXPRESSION can be any kind of boolean expression or variable WITHOUT white spaces. It is very important to respect the white spaces, otherwise the translation might not work.

The expressions need to be separated by white spaces. For example if (a>0 && b>0 || !x) then c=3 else c=1 is a valid expression, while if ( a>0&&b>0 || !x ) then c=3 else c=1 it's not.
